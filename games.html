My Board Game Catalog!

Things I like:
  • Chaos: I'm bad at imagining the game state several steps ahead (I'm terrible at chess) so I prefer games that reward flexibility. When a game has low levels of stochasticity and little player interaction, it can feel like a “better” player will almost always win. That takes some of the fun away for me, particularly when players have different comfort levels with the game.
  • Complexity: I love when there's a lot going on. I don't really get decision paralysis, so I love having a lot of interesting choices.
  • Rich storytelling: True role playing games like D&D are great, but require improv skills I'm still building (not to mention the overhead of designing a plot/character). Board games provide a structured alternative: even when Betrayal lacks strategy its an amazing storytelling vehicle, and the Betrayal Legacy rule that stipulated I must introduce each of my killer porcelain dolls by name had me lovingly doting on each of them.
This page is meant to help people choose games for game night, or help guide people looking to get a game. Some key icons:
  • : Complexity rating, taken from BoardGameGeek. I've classified games as Heavy (CR > 3), Light (CR < 3), and Party.
  • : Playtime. I'm really bad at estimating this (and it varies a lot by person/group) so just use these numbers as relative time costs
  • : Number of players I would have fun playing with
  • : Optimal number of players
